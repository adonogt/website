# Minimal Embedded Portfolio Website

This project hosts a minimalist, typography-focused website for an embedded systems developer. The layout follows the provided centered mockup and keeps every page visually consistent.

## Pages
- **Home** — Introduces Gustavo with the exact typography and spacing from the mockup.
- **Blog** — Placeholder note explaining the section is under construction.
- **Projects** — Highlights representative embedded builds and areas of focus.
- **Gear** — Lists desk, stationery, software, and routine essentials.
- **Contact** — Shares how to begin a mindful collaboration.

## Design Principles
- Clean white background with centered typography inspired by the original mockup.
- Shared navigation with a centered bullet home link and consistent spacing on every page.
- Noto Sans JP is used to support multilingual text, including Japanese characters.
- Identical paragraph spacing and alignment across the site to maintain rhythm and readability.

## Development
All pages are static HTML files styled with a single CSS file located at `assets/css/style.css`. No build tools are required—open `index.html` in a browser to view the site.
