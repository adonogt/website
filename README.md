# Minimal Embedded Portfolio Website

This project hosts a minimalist, typography-focused website for an embedded systems developer. The layout references a centered hero mockup and extends it into a fully navigable experience.

## Pages
- **Home** — Presents the core philosophy and areas of focus.
- **Blog** — Currently a placeholder announcing upcoming articles.
- **Gear** — Lists hardware, stationery, software, and well-being tools used daily.
- **About** — Shares background, experience, and guiding values.
- **Contact** — Provides guidance for reaching out and initiating collaborations.

## Design Principles
- Clean white background with centered typography inspired by the original mockup.
- Shared navigation at the top of every page with consistent spacing and responsive behavior.
- Noto Sans JP is used to support multilingual text, including Japanese characters.
- Identical paragraph spacing across the site to maintain rhythm and readability.

## Development
All pages are static HTML files styled with a single CSS file located at `assets/css/style.css`. No build tools are required—open `index.html` in a browser to view the site.
